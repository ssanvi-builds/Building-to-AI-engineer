# Building-to-AI-engineer

From military officer to AI engineer. Building projects to acquire the skills needed for a remote AI engineering role in the defense/tech dual market.

## Objective

Land a remote AI Engineer position by December 2027.

* Setting a goal requires courage because you will be able to know when you have failed. Nevertheless, only those willing to measure success and failure will be able to learn, adapt and improve eventually reaching their objective.

## Values

These principles guide every project in this repo:

**Learn by building.** No tutorial hell. Every concept gets a working project, not just notes.

**LLM use.** If I don't understand a line of code, I don't ship it. First, I look for the concept in the documentation and if it's not enough I will use AI in a socratic way. That way I force myself to understand and reason rather than just getting an answer.

**Honest scope.** I document what I don't know yet. Projects labeled "vibe-coded" are marked as such — I don't claim skills I haven't internalized.
